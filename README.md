# ☾  
![☾ Logo Code Golf](./Extra/Assets/golf1.png)

Requires python3.12

To view code you need the custom font
> [The font is in /Extra/FontCompose/Output]
To type glyphs the .XCompose file is in the same directory
  
Install: `./install.sh`  
  
Run a file: `☾ /path/to/file.☾`  

### Credits
